## 🤔 APA ITU CODING?

**Coding = Kasih Instruksi ke Komputer**

**Analogi: Resep Masakan**

```
RESEP: Nasi Goreng
1. Ambil 1 piring nasi
2. Panaskan 2 sendok minyak
3. Pecahkan 1 telur, aduk
4. Masukkan nasi, aduk 3 menit
5. Tambah kecap, aduk rata
6. Sajikan!
```

**Coding juga begitu:**

```javascript
// "Resep" untuk komputer:
1. Ambil angka 1 sampai 10
2. Untuk setiap angka, kalikan 2
3. Tampilkan hasilnya
4. Selesai!
```

**COMPUTER = Robot yang sangat patuh tapi BODOH**
- Dia lakukan PERSIS apa yang kamu suruh
- Tidak bisa mikir sendiri
- Harus jelas setiap langkah!

## 📝 ISTILAH YANG AKAN SERING MUNCUL

| Istilah | Arti Sederhana | Contoh |
|---------|----------------|--------|
| **Variable** | Tempat nyimpan data | `let nama = "Budi"` |
| **String** | Teks | `"Halo Dunia"` |
| **Number** | Angka | `42`, `3.14` |
| **Boolean** | Benar/Salah | `true`, `false` |
| **Array** | Daftar/List | `[1, 2, 3]` |
| **Object** | Benda dengan properti | `{nama: "Budi", umur: 25}` |
| **Function** | Kumpulan code yang bisa dipanggil | `function tambah(a,b) { return a+b; }` |
| **Loop** | Pengulangan | `for (let i=0; i<5; i++) {...}` |
| **If-Else** | Kondisi | `if (nilai > 70) { lulus }` |

## 🎓 CONTOH SESI BELAJAR LENGKAP

### HARI 1: Variable

**STEP 1: Baca Teori (5 menit)**

```
VARIABLE = Tempat nyimpan data

Analogi: KOTAK

┌─────────────┐
│   nama      │  ← Label kotak
│  "Budi"     │  ← Isi kotak
└─────────────┘

Cara buat:
let nama = "Budi";
  │    │    │
  │    │    └─ Isi/data
  │    └────── Sama dengan
  └─────────── Nama variable

Cara pakai:
console.log(nama);  // Output: "Budi"
```

**STEP 2: Lihat Contoh (5 menit)**

```javascript
// CONTOH 1: Variable sederhana
let nama = "Budi";
let umur = 25;
let tinggi = 170.5;

console.log(nama);      // Output: Budi
console.log(umur);      // Output: 25
console.log(tinggi);    // Output: 170.5

// CONTOH 2: Ubah isi variable
let nilai = 80;
console.log(nilai);     // Output: 80

nilai = 90;             // Ubah jadi 90
console.log(nilai);     // Output: 90

// CONTOH 3: Operasi dengan variable
let a = 10;
let b = 5;
let hasil = a + b;

console.log(hasil);     // Output: 15
```

**STEP 3: Latihan Terbimbing (10 menit)**

## 🆘 KETIKA BINGUNG

### "Saya tidak paham penjelasan ini!"

**SOLUSI:**
```
1. JANGAN lanjut ke materi berikutnya
2. Ulangi baca penjelasan
3. Copy contoh code ke console
4. RUN code-nya
5. Lihat outputnya
6. Ubah angka/values di code
7. Run lagi, lihat apa yang berubah
8. Ulangi sampai "AHA!" moment
```

### "Code saya error!"

**SOLUSI:**
```
1. BACA error message-nya
2. Biasanya ada petunjuk baris berapa
3. Cek typo (huruf besar/kecil beda!)
4. Cek titik koma (;)
5. Cek kurung buka/tutup
6. Google error message
7. Tanya mentor/teman
```

### "Saya lupa syntax!"

**SOLUSI:**
```
NORMAL! Semua developer juga lupa!

CARA:
1. Jangan hafal!
2. Buka catatan/file contoh
3. Copy syntax yang benar
4. Modifikasi sesuai kebutuhan
5. Lama-lama akan hafal sendiri
```
