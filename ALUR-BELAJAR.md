## 🆘 TROUBLESHOOTING

### MASALAH 1: "Tidak paham penjelasan di 30-day-challenge.js"

```
SOLUSI:
1. JANGAN PANIK! Ini normal.
2. Buka cepat-paham.md
3. Ctrl + F → Cari topik yang tidak dipahami
4. Baca penjelasan lebih detail
5. Lihat contoh code tambahan
6. Copy contoh, paste di console
7. Run, lihat hasil
8. Kembali ke 30-day-challenge.js
9. Lanjut latihan
```

### MASALAH 2: "Code saya error terus!"

```
SOLUSI:
1. BACA error message di console
2. Biasanya ada petunjuk baris berapa
3. Cek typo (huruf besar/kecil beda!)
4. Cek titik koma (;)
5. Cek kurung buka/tutup
6. Google error message
7. Buka cepat-paham.md, cari syntax terkait
8. Compare code Anda dengan contoh
9. Tanya mentor/teman jika masih stuck
```

### MASALAH 3: "Tidak ada waktu 20 menit!"

```
SOLUSI:
- BUAT waktu! 20 menit = 1 episode Netflix
- BANGUN 20 menit lebih awal
- KURANGI scroll media sosial 20 menit
- INGAT: 20 menit × 30 hari = 10 jam = TRANSFORMASI!
- Jika benar-benar tidak ada: 10 menit saja!
  Yang penting CODE setiap hari!
```

### MASALAH 4: "Merasa bodoh, tidak progress"

```
SOLUSI:
1. BUKA START-HERE.md
2. Baca bagian "AFFIRMASI UNTUK DEVELOPER"
3. BUKA 30-day-challenge.js Hari 1
4. Lihat: Anda SUDAH bisa variable!
5. BANDINGKAN dengan diri sendiri, bukan orang lain
6. INGAT: Setiap expert pernah beginner
7. ISTIRAHAT 1-2 hari jika perlu
8. KEMBALI dengan fresh mind
```
