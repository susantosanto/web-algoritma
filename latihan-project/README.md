## 🎯 CARA MENGGUNAKAN

### **Step 1: Pilih Project**
Pilih project yang ingin dikerjakan. Mulai dari level Pemula.

### **Step 2: Baca Blueprint**
Buka file `BLUEPRINT.md` di folder project yang dipilih.

### **Step 3: Ikuti 7 Langkah Universal**
Gunakan framework dari `ALGORITMA-PEMECAHAN-MASALAH.md`:
1. Pahami masalahnya
2. Bedah masalahnya
3. Tentukan strategi
4. Tulis pseudo-code
5. Uji manual
6. Tulis kode
7. Verifikasi

### **Step 4: Isi Kode Sendiri**
Setiap blueprint punya comment `// TULIS KODEMU DI SINI` atau `<!-- ISI DI SINI -->`.
Kamu mengisi sendiri sesuai petunjuk.

### **Step 5: Minta Bantuan AI**
Jika stuck, tanya AI:
- "Saya sedang mengerjakan [nama project]"
- "Saya sudah sampai tahap [tahapan]"
- "Saya bingung di bagian [bagian yang bingung]"
- "Ini kode saya [tempel kode]"
- "Tolong bantu jangan kasih jawaban, tapi kasih petunjuk"

## 🛠️ TEKNOLOGI YANG DIPAKAI

### **Frontend**
- **HTML5**: Struktur halaman
- **Tailwind CSS**: Styling (utility-first)
- **JavaScript**: Logika dan interaktivitas

### **Tools**
- **Browser**: Chrome/Firefox untuk testing
- **VS Code**: Code editor
- **Live Server**: Extension untuk auto-reload

### **Cara Setup**
```
1. Buat folder baru untuk project
2. Buat file: index.html, style.css (opsional), app.js
3. Buka dengan VS Code
4. Pasang Live Server extension
5. Klik "Go Live" untuk menjalankan
```

## 💡 TIPS SUKSES

1. **Mulai dari yang Sederhana** - Jangan langsung bikin fitur kompleks
2. **Test Sering-Sering** - Setelah ubah kode, cek di browser
3. **Baca Error Message** - Error adalah guru, bukan musuh
4. **Jangan Takut Salah** - Refactor (perbaiki) itu bagian dari proses
5. **Gunakan AI untuk Belajar** - Tanya "mengapa" bukan "apa jawabannya"
