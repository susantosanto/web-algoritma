#### 1. Mantra String Lengkap

    *   **Kegunaan:** Mengembalikan iterator dari semua hasil pencocokan regex.

    *   **Kegunaan:** Menghapus spasi dari awal string.

    *   **Kegunaan:** Menghapus spasi dari akhir string.

### Mantra #1: `.map()` - Mantra Transformasi (The Transfiguration Spell)

**Model Mental: Pabrik Ajaib**

Bayangkan sebuah pabrik ajaib. Anda memasukkan **100** kentang mentah ke satu sisi. Di dalam pabrik (`.map`), ada satu instruksi: "Goreng setiap item". Dari sisi lain, keluar **100** kentang goreng.

*   **Karakteristik Utama:** Jumlah item yang masuk **SELALU SAMA** dengan jumlah item yang keluar. Yang berubah hanyalah **BENTUK** atau **ISI** dari setiap item.
*   **Rumus Sihir:** `Array Lama (Jumlah N)` -> `Array Baru (Jumlah N)`

**Rapalan Mantra (The Incantation):**

```javascript
// "Untuk setiap 'item' di dalam array, kembalikan 'item' dalam bentuk baru"
arrayAsli.map(item => item_yang_sudah_diubah);
```

**Lihat Sihirnya (See it in Action):**

```javascript

```

**Jejak Eksekusi:**

### Mantra #2: `.filter()` - Mantra Seleksi (The Banishment Spell)

**Model Mental: Gerbang Penjaga**

Bayangkan sebuah gerbang ajaib yang dijaga oleh golem. Golem tersebut hanya mengizinkan item yang memenuhi syarat tertentu untuk lewat. Anda mengirim **100** buah-buahan (sebuah array) ke gerbang. Perintah golem (`.filter`) adalah: "Hanya izinkan 'apel' yang lewat". Mungkin hanya **30** apel yang berhasil lewat, sisanya (jeruk, pisang) "dibuang" atau ditolak.

*   **Karakteristik Utama:** Jumlah item yang keluar **MUNGKIN LEBIH SEDIKIT** atau sama dengan jumlah item yang masuk. Tidak pernah lebih banyak. Item yang lolos seleksi tidak berubah bentuk.
*   **Rumus Sihir:** `Array Lama (Jumlah N)` -> `Array Baru (Jumlah <= N)`

**Rapalan Mantra (The Incantation):**

```javascript
// "Untuk setiap 'item', jika 'kondisi' terpenuhi (true), simpan item ini. Jika tidak (false), buang."
arrayAsli.filter(item => kondisi_yang_harus_benar);
```

**Lihat Sihirnya (See it in Action):**


**Jejak Eksekusi:**
1.  `filter` mengambil `120`. Kondisi `120 > 100` adalah `true`. `120` disimpan.
2.  `filter` mengambil `80`. Kondisi `80 > 100` adalah `false`. `80` dibuang.
3.  `filter` mengambil `250`. Kondisi `250 > 100` adalah `true`. `250` disimpan.
4.  ...dan seterusnya.

**Uji Coba Mantra Anda:**

### Mantra #3: `.reduce()` - Mantra Pemadatan (The Essence Extraction Spell)

**Model Mental: Kuali Alkemis**

Ini adalah mantra yang paling kuat dan serbaguna. Bayangkan sebuah kuali alkemis (`.reduce`). Anda harus memulai dengan "cairan dasar" (nilai awal). Kemudian, Anda memasukkan bahan-bahan (elemen array) ke dalam kuali, **satu per satu**. Setiap kali bahan baru masuk, ia bereaksi dengan cairan di dalam kuali dan menghasilkan cairan baru. Proses ini berlanjut sampai semua bahan habis, dan yang tersisa di kuali adalah **satu esensi akhir** (satu nilai tunggal).

*   **Karakteristik Utama:** Mengubah seluruh array menjadi **SATU NILAI TUNGGAL**. Nilai ini bisa berupa angka, string, objek, atau bahkan array baru.
*   **Rumus Sihir:** `Array Lama (Jumlah N)` -> `Satu Nilai Akhir`

**Rapalan Mantra (The Incantation):**

```javascript
arrayAsli.reduce((cairanDiKuali, bahanBerikutnya) => {
  // Logika reaksi antara cairan dan bahan baru
  return cairan_hasil_reaksi;
}, cairanDasar); // <-- Cairan dasar SANGAT PENTING!
```

**Lihat Sihirnya (See it in Action):**
Mari selesaikan tugas ketiga Raja: menghitung total pendapatan.

```javascript
const penjualanUSD = [10, 20, 5, 30];

// cairanDiKuali kita sebut 'total'
// bahanBerikutnya kita sebut 'penjualan'
// cairanDasar kita adalah 0
const totalPendapatan = penjualanUSD.reduce((total, penjualan) => {
  return total + penjualan;
}, 0);

console.log(totalPendapatan); // Output: 65
```

**Jejak Eksekusi (Alur dalam Kuali):**
*   **Awal:** `total` = `0` (cairan dasar).
*   **Iterasi 1:** `bahanBerikutnya` = `10`. Reaksi: `0 + 10`. Kuali sekarang berisi `10`.
*   **Iterasi 2:** `bahanBerikutnya` = `20`. Reaksi: `10 + 20`. Kuali sekarang berisi `30`.
*   **Iterasi 3:** `bahanBerikutnya` = `5`. Reaksi: `30 + 5`. Kuali sekarang berisi `35`.
*   **Iterasi 4:** `bahanBerikutnya` = `30`. Reaksi: `35 + 30`. Kuali sekarang berisi `65`.
*   **Akhir:** Semua bahan habis. Esensi akhir yang tersisa adalah `65`.

**Jebakan Umum:**
Kesalahan paling umum bagi penyihir pemula adalah **lupa memberikan `cairanDasar` (nilai awal)**. Ini bisa menyebabkan hasil yang tidak terduga, terutama pada array yang kosong. Selalu berikan nilai awal!

**Uji Coba Mantra Anda:**

```javascript

    // Kuali 1: Menghitung total pendapatan dari penjualan bernilai tinggi
    const pendapatanBernilaiTinggi = penjualanBernilaiTinggi.reduce((total, p) => total + p.total, 0);

    // Kuali 2: Mengelompokkan penjualan berdasarkan wilayah
    const penjualanPerWilayah = penjualanDenganTotal.reduce((kuali, penjualan) => {
      const wilayah = penjualan.wilayah;
      // Jika wilayah belum ada di kuali, buat dengan nilai 0
      if (!kuali[wilayah]) {
        kuali[wilayah] = 0;
      }
      // Tambahkan total penjualan ke wilayah yang sesuai
      kuali[wilayah] += penjualan.total;
      return kuali; // Kembalikan kuali untuk iterasi berikutnya
    }, {}); // Cairan dasar adalah objek kosong {}
    console.log(" -> Statistik dihitung.\n");

    // --- Tahap 4: Mantra Penantian (async/await) ---
    console.log("Langkah 4: Menyimpan laporan...");
    const laporanFinal = {
      pendapatanPenting: pendapatanBernilaiTinggi,
      rekapWilayah: penjualanPerWilayah
    };
    await simpanKeSpreadsheet(laporanFinal);
    console.log(" -> Penyimpanan selesai.\n");

    // --- Tahap 5: Menampilkan Hasil Akhir ---
    console.log("--- RINGKASAN LAPORAN AKHIR ---");
    console.log(`Total Pendapatan dari Penjualan Bernilai Tinggi: $${pendapatanBernilaiTinggi}`);
    console.log("Rekapitulasi Pendapatan per Wilayah:");
    // Loop melalui objek rekap wilayah
    for (const wilayah in penjualanPerWilayah) {
      console.log(`  - ${wilayah}: $${penjualanPerWilayah[wilayah]}`);
    }
    console.log("\nRitual Pembuatan Laporan Selesai dengan Sempurna!");

  } catch (error) {
    console.error("Sebuah kesalahan terjadi selama ritual:", error);
  }
}
```
